# MarketPy User Guides

## Strategy Lab
- Open the Strategy Lab canvas.
- Add signal/indicator/operator/action blocks.
- Connect blocks and validate the strategy graph.
- Run instant backtest and review equity curve, metrics, and trades.

## Alerting Hub
- Create alerts with condition type: price, volume, or volatility.
- Choose operator and threshold.
- Configure cooldown and channels: webhook, Telegram, Discord, email.
- Save and monitor triggered alerts.

## Risk Cockpit
- VaR: review 95%/99% VaR and threshold warnings.
- Stress testing: compare predefined and custom scenario outcomes.
- Correlation heatmap: inspect pairwise relationships and shift highlights.

## Model Governance
- Register model versions in the registry.
- Promote selected version to production.
- Track drift metrics and alerts.
- Review feature importance trends and shifts across versions.
